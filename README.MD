## Pré-requisitos:
### Dependências necessárias:
- Docker Desktop
- Docker Compose
- Git Bash

## Como Rodar:

### Repositório:
Clone o seguinte repositório inserindo este comando no terminal Bash:

```sh
git clone https://github.com/Viniciusgigante19/tf-BlogFlow.git
```

### Caminho do repositório:
Rode no mesmo terminal bash já aberto:

```sh
cd ./tf-BlogFlow
```

### Projeto no Ar:
No caminho do passo passado rode:

```sh
docker compose up --build
```

### URL do projeto:
Após o build dos conteineres acesse em seu navegador:

http://localhost:8080/

### URL da API:
http://localhost:8080/api/posts
